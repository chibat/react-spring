# ローカル環境でのアプリケーションの起動

お好みのIDEで Spring Boot アプリケーションの起動を実行してください。

Gradle から実行する場合は、以下のコマンドになります。
```
./gradlew bootRun
```

開発用フロントエンドサーバの起動

```
npm start
```
