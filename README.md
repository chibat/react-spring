
# React x Spring type-safe communication

## Prerequisites
* JDK: 17+
* Node.js: 16+ 

## Local environment

```
./gradlew bootRun
```

```
$ cd frontend
$ npm install
$ npm start
```

Open http://localhost:3000/ in your Web Browser

## Build for Production

```
$ ./gradlew bootJar
$ java -jar build/libs/react-spring-0.0.1-SNAPSHOT.jar
```
